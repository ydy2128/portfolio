History

개발자로서 발전되어가는 제 역사를 소개합니다.

2017.03 ~ 2017.06

Visual Studio 툴과 C# 언어를 이용하여 식당에서 사용할 수 있는 주문&계산 프로그램을 혼자 만들었습니다. 윈도우 폼의 디자인을 신경써서 만들었습니다. 테이블이 나열돼 있는 메인 화면에서 테이블을 선택하면 주문 가능한 메뉴들이 나열돼 있으며 주문한 메뉴들을 관리할 수 있고 주문된 메뉴 개별 금액과 총액이 적혀있습니다. 테이블 이동/계산/주문 취소 등도 할 수 있습니다. 메인 화면에도 테이블 별로 주문된 메뉴와 금액들이 확인 가능합니다. 그리고 계산 이력들을 확인할 수 있습니다. (인쇄 기능을 이 프로그램에 추가했는지 다른 C# 프로젝트에 추가했는지 헷갈립니다...) 프로젝트를 완성하면 느껴지는 뿌듯함이 너무 좋아서 개발에 흥미가 생기게 됐습니다.

2017.08 ~ 2017.11

html과 javascript, css 만을 이용하여 서버 사용 없이 처음으로 웹페이지를 팀원들과 함께 구현했습니다. CGV 영화 홈페이지를 모방하여 만들었습니다. 데이터베이스 사용 없이 오로지 프론트엔드를 구현한 것이기 때문에 디자인과 페이지 이동 등을 비슷하게 모방했습니다. 마우스를 올릴 때 또는 클릭할 때 변화되는 것들과 객체들의 배치 등을 많이 신경써서 만들었습니다. 처음으로 팀 리더가 되어서 팀원 관리 능력이 중요하다는 것을 깨닫게 되었으며 협동심도 배웠습니다. 열심히 해준 팀원들 덕분에 팀프로젝트가 즐거웠습니다.

2018.03 ~ 2018.06

Android Studio 툴과 java 언어를 이용하여 자기계발 앱을 만들었습니다. 공부하는 시간을 타이머를 통해서 측정할 수 있고 기록할 수 있습니다. (기록된 화면이 달력이었나... 기억이 잘 안 납니다...공부시간 따로 운동시간 따로 기록했던 것 같기도...) 바탕화면으로 나와도 floating 버튼을 이용하여 타이머를 control할 수 있습니다. 그리고 또 다른 메뉴에는 단계별로 학습(기억이 잘 안 납니다ㅠ)같은 것을 할 수 있게 돼있습니다. 팀원은 대체로 버그를 찾고 에러 해결법을 찾는 역할을 맡았습니다.

2018.08 ~ 2018.11

Unity 툴과 C# 언어를 이용하여 크레이지 아케이드 게임을 모방하여 만들었습니다. 짧은 시간 안에 거의 혼자 완성한 것이라 기억에 많이 남습니다. (다른 팀원이 바닥 위에 캐릭터를 올려 놓고 스페이스바를 누르면 물풍선이 나타나는 것까지 구현했습니다.) single play이며 3라운드까지 있습니다. 방향키로 캐릭터 이동이 가능하며 스페이스바로 물풍선을 생성할 수 있습니다. 바닥은 바둑판 처럼 칸들로 구성되어 있으며 캐릭터가 서 있는 칸 위에 물풍선이 생성됩니다. 그리고 한 칸에는 오로지 한 물풍선만 생성되며 생성된 물풍선은 3초 뒤에 십자가 모양인 물줄기가 잠시 나오면서 터집니다. 칸마다 장애물 또는 물줄기에 의해 제거되는 상자가 놓여 있기도 합니다. 상자가 제거된 후 그 칸에 아이템이 나올지는 랜덤으로 설정했습니다. 아이템은 '캐릭터 속도향상/물줄기길이 한 칸 늘리기/생성할 수 있는 물풍선 개수 하나 늘리기'로 구성돼 있습니다. 적은 player에게 다가오게 돼있으며, player는 적과 닿거나 물줄기에 닿으면 게임은 실패로 끝납니다. 방향키 누를 때의 캐릭터가 달리는 모션과 물풍선 효과 등(무료에셋에서 구함)도 구현했으며 게임 시작 화면, 다음 라운드로 넘어가는 화면, 실패했을 때의 화면, 성공했을 때의 화면도 구현했습니다. 프로젝트 완성 후에도 게임이 재밌어서 play를 많이 했던 기억이 있습니다.
